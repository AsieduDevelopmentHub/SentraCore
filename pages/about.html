About SentraCore

SentraCore is a system intelligence stack: a desktop-first experience backed by a small, fast API and a deployable engine—aimed at monitoring, clarity, and automation.

What it is

SentraCore brings together real-time system visibility and a headless control plane. The Flutter dashboard focuses on operator UX; the Python engine handles workloads and exposes a consistent HTTP surface for scripts and integrations.

Architecture

Three layers work together but stay separable: a frontend (Flutter desktop), a runtime engine (Python, packaged for distribution), and an API layer (FastAPI / Starlette) for health, status, and extension points.

Tech stack

  • FastAPI — modern async API, OpenAPI-friendly.
  • Flutter — cross-platform UI for Windows, macOS, and Linux.
  • PyInstaller — one-file style engine builds for CI and releases.

Design goal

Ship a monitoring and automation engine that feels intentional: low ceremony to install, predictable to operate, and straightforward to embed in your own workflows.
