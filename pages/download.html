Quick start

  1. Click the button for your OS—your browser should download the file from GitHub’s CDN.
  2. If a button opens the release page instead, your browser blocked the API call—use the assets list there.
  3. Run the installer or unpack the archive, then launch the dashboard.

Filenames follow CI output (e.g. SentraCore_Setup_v…, SentraCore_macOS_…, SentraCore_linux_…).
