Windows

Graphical installer that places the dashboard and engine under the folder you select. Shortcuts are added for quick access. Run the installer once; after that, launch SentraCore from the Start menu or desktop if you opted in.

macOS

Archive containing the application bundle. Unzip, move the app where you keep tools, and open it. On first launch, macOS may ask for confirmation for software from an unidentified developer—this is normal for ad-hoc signed builds until you notarize or adjust Gatekeeper policy.

Linux

Self-contained AppImage for x86_64. Mark the file executable, then run it from a terminal or file manager. Pin it to your dock or application menu using your desktop environment’s tools if you like.

Download for Linux

First run

  1. Download the build for your system and complete any platform prompts.
  2. Open the SentraCore dashboard. The engine starts or connects in the background as described in the docs.
  3. Allow local network access for localhost if a firewall prompts you—that is how the UI talks to the engine.

File names follow the naming used in automated builds. If something looks unfamiliar, compare sizes and dates on the release page before opening executables.

Updating

New versions appear on the same Releases page. Download the new installer or archive for your platform, install over or replace the previous copy, and launch again. Your user-level configuration paths are designed to survive upgrades; if you customized install locations, keep notes so you know where to look after an update.
